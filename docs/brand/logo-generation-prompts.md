# Argus Logo Generation Prompts

Date: 2026-04-19

This document provides controlled prompt variants for exploring the Argus logo direction without drifting away from the approved brand brief.

Use these prompts alongside [argus-logo-brand-spec.md](/Users/yann.moren/vision/docs/brand/argus-logo-brand-spec.md).

## Base Constraints For Every Prompt

Apply these constraints to every exploration pass:

- premium enterprise software identity
- dark-mode native
- abstract eye plus network or radar symbolism
- cerulean intelligence core
- restrained violet perimeter shift
- clean, geometric, scalable icon
- `Argus` visually dominant
- `THE OMNISIGHT PLATFORM` clearly secondary
- no gaming or esports tone
- no sparkle artifacts
- no excessive neon
- no cluttered composition

## Shared Negative Prompt

```text
generic startup logo, gaming logo, esports logo, cyberpunk excess, heavy neon, cartoonish eye, too many details, overcomplicated shards, lens flare, sparkle icon, watermark, busy background, cheap 3D effects, glossy plastic, illegible typography, weak hierarchy, flat boring icon, clipart, stock logo
```

## Prompt A: Master Refinement Pass

Use this prompt to get closest to the intended final logo.

```text
Refine a premium enterprise logo for “Argus | The OmniSight Platform” on a deep obsidian-to-charcoal background. Create a centered logo lockup with a clean abstract icon that combines a vigilant eye with an omnidirectional radar or network hub. The symbol must feel geometric, precise, refined, and scalable, with a cerulean blue intelligence core and a restrained violet perimeter glow. The icon should be slightly simplified for better clarity and small-size legibility.

Typography must be premium and modern. “Argus” should be large, bold, luminous off-white, and clearly dominant. Add a subtle vertical separator. “THE OMNISIGHT PLATFORM” should be smaller, lighter, and clearly secondary. The result should feel like a real master brand mark for a high-end B2B vision intelligence platform. Keep the background dark, matte, and elegant. Avoid flashy neon or consumer-tech styling.
```

## Prompt B: Icon Simplification Pass

Use this prompt when the icon is still too busy.

```text
Design a simplified premium logo icon for “Argus | The OmniSight Platform” that fuses a watchful eye with a radar or network hub. Focus on silhouette clarity, icon scalability, and geometric precision. Use fewer internal overlaps and fewer thin protrusions. Preserve the central cerulean intelligence node and subtle violet outer shift, but minimize visual clutter. The symbol must still feel advanced, elegant, and unmistakably vigilant. Pair it with a strong “Argus” wordmark and a clearly secondary platform descriptor.
```

## Prompt C: Typography Hierarchy Pass

Use this when the symbol is working but the lockup still feels generic.

```text
Create a premium dark-mode technology logo for “Argus | The OmniSight Platform” with special attention to typography hierarchy. The icon should be a refined geometric eye-network mark with cerulean and violet luminescence, but the main focus is making “Argus” feel distinctive, bold, elegant, and enterprise-grade. The vertical separator should be subtle. “THE OMNISIGHT PLATFORM” must be lighter, smaller, and structurally secondary. The overall result should feel like a sophisticated flagship platform identity, not a generic startup wordmark.
```

## Prompt D: Product UI Variant Pass

Use this for flatter, cleaner in-product usage.

```text
Create a product-ready logo for “Argus | The OmniSight Platform” designed for dark enterprise software UI. Use a clean abstract eye-network symbol with minimal lighting effects, limited depth, and restrained glow. The palette should remain cerulean to violet with luminous off-white typography, but the result must feel operational, crisp, and durable inside real application chrome. Avoid heavy rendering, deep bevels, and hero-style presentation. This should be the clean UI lockup, not the marketing hero version.
```

## Prompt E: Hero Branding Pass

Use this for a high-impact marketing render while staying within the brand.

```text
Create a premium hero-brand logo presentation for “Argus | The OmniSight Platform” on a deep obsidian matte background. The icon should be a refined eye-network fusion with a luminous cerulean core and subtle violet atmosphere, suggesting digital bioluminescence and constant intelligent awareness. The lockup should feel polished, elegant, and cinematic, but still disciplined enough for enterprise B2B branding. Keep the word “Argus” dominant and the platform descriptor secondary. Avoid turning the piece into neon sci-fi or gaming artwork.
```

## Prompt F: Monochrome / Hardware Pass

Use this to test whether the logo survives without color.

```text
Design a monochrome version of the “Argus | The OmniSight Platform” logo optimized for hardware labeling, embossing, print, and small-scale reproduction. The icon should remain a clear eye-network symbol with excellent silhouette clarity and readable internal geometry, even without glow or gradient. The typography must preserve hierarchy, with “Argus” dominant and the platform descriptor secondary. The result should feel premium, minimal, and fabrication-ready.
```

## Recommended Exploration Order

Run the prompts in this order:

1. Prompt A for the closest final-direction result
2. Prompt B if icon clarity is still weak
3. Prompt C if the wordmark feels too generic
4. Prompt D for the in-product variant
5. Prompt E for marketing usage
6. Prompt F to validate durability without effects

## Selection Criteria

When reviewing generations, prefer the version that:

- reads fastest at a glance
- has the cleanest icon silhouette
- feels most premium and least flashy
- gives `Argus` the strongest visual authority
- keeps the tagline secondary
- would still work when flattened into a vector
